# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built with **C# Windows Forms (.NET Framework)**.

## 📌 Overview

This project is a classic two-player Tic Tac Toe game featuring score tracking, winner detection, draw detection, and a user-friendly graphical interface.

## ✨ Features

- 👥 Two-player mode (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 📊 Scoreboard system
  - Player X Wins
  - Player O Wins
  - Draw Count
- 🔄 Restart current game
- ♻️ Reset all scores
- 🎨 Winning row, column, or diagonal highlighted in green
- 🔒 Prevents selecting an already occupied cell
- 🔔 Winner notification using MessageBox
- 🌐 Social media buttons (GitHub, LinkedIn, X/Twitter)

## 🛠️ Technologies Used

- C#
- Windows Forms (WinForms)
- .NET Framework
- Visual Studio

## 🧠 Game Logic

The game checks all possible winning conditions:

### Rows
- First row
- Second row
- Third row

### Columns
- First column
- Second column
- Third column

### Diagonals
- Main diagonal
- Anti-diagonal

When a winning condition is met:

1. The winning cells are highlighted.
2. The winner score is increased.
3. A winner message is displayed.
4. The board resets automatically.
